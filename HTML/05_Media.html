Image

The <img> tag is used to embed an image in an HTML page. The src attribute specifies the path to the image,l and the alt attributee provides alternative text for the image.

Video

The <video> tag is used to embed a video in an HTML page. The controls attribute adds video controls, like play, pause,e and volume. The <source> elements specify multiple video formats for better browser compatibility.

Audio

The <audio> tag is used to embed an audio file in an HTML page. The controls attribute adds audio controls, like play, pause, and volume. The <source> elements specify multiple audio formats for better browser compatibility.

Iframe

The <iframe> tag is used to embed another HTML page within the current page. The src attribute specifies the URL of the page to embed.

Embed

The <embed> tag is used to embed external content or plugins, such as Flash content. The src attribute specifies the path to the content.

Object

The <object> tag is used to embed multimedia objects, such as PDFs or Flash content. The data attribute specifies the path to the object.
